## Installation

1. Add this repository to your Home Assistant Supervisor:
   - Settings ‚Üí Add-ons ‚Üí Add-on Store ‚Üí ‚ãÆ ‚Üí Repositories

3. Configure your Todoist API token

4. Start the add-on
